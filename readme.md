## ‚ö°Ô∏è Git Extensions
*Git is designed for source control management. Extensions help people to extend the idea and push version control to everywhere*

- [Git Large File Storage](https://git-lfs.github.com/) ‚Äî practical solution for versioning large files. supported by GitHub
- [Git Virtual File System or GVFS](https://github.com/Microsoft/GVFS) ‚Äî solution for managing very large Git repository while maintaining speed and efficiency of most operations, in development by Microsoft.
- [git-annex](https://git-annex.branchable.com/) ‚Äî allow managing massive binaries among machines, as if operating a standard git repository. Possible to create a synchronized folder with [git-annex assistant](https://git-annex.branchable.com/assistant/)
- [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) - A Visual Studio code extension which helps you visualize code authorship at a glance via Git blame annotations and code lens. Allows you to seamlessly explore Git repositories and gain valuable insights via powerful comparison commands.

## ‚ö°Ô∏è Git Repository Hosting
*People have plenty of options to host their source code*

- [GitHub](https://github.com/) ‚Äî the de-facto git hosting service. Perfect integration with most external services.
- [GitLab.com](https://about.gitlab.com/gitlab-com/) ‚Äî a free Git repository hosting service served by GitLab EE. Unlimited repositories and private collaborators
- [BitBucket](https://bitbucket.org/) ‚Äî well-known for its free private repository (5 user max).
- [Kiln](https://www.fogcreek.com/kiln/) ‚Äî paid Git repository hosting service
- [CodePlex](https://www.codeplex.com/) ‚Äî Microsoft‚Äôs free open source code hosting service with many ASP/C# OSS projects
- [AWS CodeCommit](https://aws.amazon.com/codecommit/) ‚Äî a SaaS service provided by Amazon Web Service on high availability infrastructure
- [Codeplane](https://codeplane.com/) ‚Äî a paid Git repository hosting service with no contributor limit
- [Deveo](https://deveo.com/) ‚Äî a paid repository hosting service with support for Git, Subversion, Mercurial, WebDAV
- [SourceForge](https://sourceforge.net) ‚Äî a long-standing free Git repository hosting service
